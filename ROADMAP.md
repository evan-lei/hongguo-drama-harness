# Skill 路线图

## 已完成 ✅

| Skill | 用途 | 触发场景 |
|-------|------|----------|
| `hongguo-platform` | 红果平台投稿规范、格式、政策、避坑 | 投稿、审核、格式、平台政策相关 |
| `drama-prompt-enrich` | 碎片想法 → 结构化故事初稿 | 有新创意、需要丰富 prompt、写故事核 |

## 规划中 📋

| Skill | 用途 | 依赖 |
|-------|------|------|
| `drama-outline` | 故事初稿 → 故事大纲 + 分集大纲（含卡点） | `drama-prompt-enrich` 产出 |
| `drama-episode` | 分集大纲 → 标准格式分集剧本 | `drama-outline` 产出 |
| `drama-qc` | 剧本质检 checklist（钩子、反差、逻辑、格式） | 任意阶段稿件 |
| `drama-genre` | 题材库 + 爆款结构模式（赘婿/重生/复仇等） | 选题阶段 |
| `drama-pull` | 拉片拆解辅助（三层次法） | 学习/Reference 阶段 |

## 工作流原则

1. **分层产出**：初稿 → 大纲 → 分集稿，不跳步
2. **人机分工**：AI 扩写与润色，人把控故事核与关键转折
3. **版本留存**：`projects/<剧名>/` 下保留每版迭代记录
4. **先大纲后全稿**：新手优先投故事大纲试水，拿编辑反馈再深化
