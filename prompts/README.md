# Prompt 片段库

存放可复用的 Prompt 模块，按创作阶段分类。完整工作流见各 Skill。

## 目录规划

```
prompts/
├── enrich/       # 故事初稿丰富（见 drama-prompt-enrich skill）
├── outline/      # 大纲生成（待 drama-outline skill）
├── episode/      # 分集剧本（待 drama-episode skill）
└── qc/           # 质检（待 drama-qc skill）
```

当前阶段以 Skill 内嵌模板为主；后续可将高频 Prompt 抽取到此目录做版本管理。
