# 红果短剧剧本格式参考

## 大纲阶段标注示例

```
【开场：九天玄女因话痨被罚下诛仙台，魂穿林府真千金】
【转折：需代替真千金扭转悲剧才能重返天庭】
【卡点：哥哥听见女主心声，得知养妹要全家性命】
```

## 分集剧本示例（节选）

```
第1集

1-1 日 内 九重天
人物：林雨欣、天将×2

△诛仙台边，云雾缭绕。林雨欣被两名天将押到台前。
天将甲：九天玄女，你因多言触犯天条，判你下凡历劫！
林雨欣（错愕）：我就多说了两句……
△林雨欣被推下诛仙台，画面白闪。

1-2 日 内 林府闺房
人物：林雨欣

△林雨欣猛然睁眼，环顾四周古色古香的房间，一脸懵。
林雨欣（喃喃）：这是……哪儿？
△铜镜中映出一张陌生少女的脸。
林雨欣：我不是玄女吗？怎么变成这样了？！
【卡点：门外传来急促脚步，有人高喊「大小姐醒了」】
```

## 场次编号规则

- `集号-场号`：`1-1` = 第 1 集第 1 场
- 同集切场景：`1-1` → `1-2` → `1-3`
- 闪回场次可单独编号或在同场内用【闪回】包裹

## 台词写作要点

1. **推动剧情** — 每句话要么揭露信息、要么制造冲突
2. **揭示人物** — 语气、用词体现身份和性格
3. **制造悬念** — 话说一半、答非所问、突然打断
4. **避免** — 「我觉得」「其实吧」等拖沓口语（除非人设需要）

## 动作描写要点

```
❌ 林雨欣心里很愤怒，觉得委屈极了。
✅ △林雨欣攥紧拳头，指甲掐进掌心，眼眶发红。

❌ 他在房间里发泄情绪。
✅ △他一把扫落桌上茶杯，碎片溅到脚边，他浑然不觉。
```
